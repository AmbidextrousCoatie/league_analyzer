# Team vs Team Colored Circles Fix

## Problem
The team vs team comparison table was missing the colored circles for team names in the first column. These circles are generated by the `teamColorMap` system and are widely used throughout the application (e.g., in league results match day tables).

## Root Cause
The JavaScript code in `components/tables.html` was only checking for the English title "Team vs Team Comparison" to trigger the special colored circle rendering, but the table title was internationalized and could be in German ("Team vs Team Vergleichsmatrix").

## Solution
Updated the JavaScript detection logic to check for both English and German versions of the title.

### **File**: `app/templates/components/tables.html`

#### **Before**:
```javascript
if (data.title && data.title.includes('Team vs Team Comparison') && groupIndex === 0 && index === 0) {
```

#### **After**:
```javascript
if (data.title && (data.title.includes('Team vs Team Comparison') || data.title.includes('Team vs Team Vergleichsmatrix')) && groupIndex === 0 && index === 0) {
```
